# 🌤️ Advanced Weather App with Voice Feedback

This is a Python-based GUI Weather App using `Tkinter`, `OpenWeatherMap API`, and `pyttsx3` for voice feedback.  
It also gives you friendly weather advice like "Nice day to go outside" or "Not a great day for outing".

---


## ⚙️ How to Run

1. Make sure Python 3.x is installed.
2. Open terminal in the `weather_app` folder.
3. Install the required packages:

```bash
pip install requests pillow pyttsx3
```

4. Run the app:

```bash
python weather_app.py
```

---

## 🔑 Required API Key

- Sign up at [https://openweathermap.org/](https://openweathermap.org/)  
- Get your free API key and paste it in the code where it says:

```python
API_KEY = "your_api_key_here"
```

---

## 🔉 Features

- Voice output for weather info.
- Simple GUI using Tkinter.
- Automatically speaks suggestions like:
  - “It’s a nice day to go outside.”  
  - “Not a great day for outing. Stay safe!”
